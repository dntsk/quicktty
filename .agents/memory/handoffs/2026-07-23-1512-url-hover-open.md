## Важный контекст

- Accepted `open_url` принадлежит app callback-context lifetime и переживает закрытие исходной pane; delivery после app shutdown отбрасывается.
- Любой URL scheme разрешён; no-scheme обрабатывается как standardized file URL с expansion `~`; text kind предпочитает default editor.
- Ошибка `NSWorkspace` логируется без URL, path, query, token и `localizedDescription`; public diagnostic содержит только `NSError` domain/code.
- Cursor state принадлежит конкретной surface view; `NSCursor.push/pop/set` не используются.
- Не коммитить/push, не менять Vendor/pin, Search, shortcut registry или config grammar.
